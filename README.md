# 💳 Bankist App (JavaScript Implementation)

This project is a banking application built as part of a JavaScript learning experience. The HTML and CSS were pre-designed, while the full application logic and interactivity were implemented using JavaScript.

## 📌 Features
- User login authentication
- Display of account balance
- Transaction history (deposits & withdrawals)
- Money transfer between accounts
- Loan request functionality
- Account closure feature
- Sorting of transactions
- Logout timer
